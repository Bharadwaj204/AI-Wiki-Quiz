# Screenshots Documentation

## How to Take Screenshots

To fulfill the submission requirements, you'll need to take screenshots of the following:

1. **Quiz Generation Page (Tab 1)**
   - Open your browser and navigate to http://localhost:3000
   - Ensure the backend is running at http://localhost:8000
   - Take a screenshot of the main quiz generation interface

2. **History View (Tab 2)**
   - Click on the "Quiz History" tab
   - Take a screenshot of the table showing previously generated quizzes

3. **Details Modal**
   - Click the "Details" button for any quiz in the history
   - Take a screenshot of the modal displaying the full quiz

## Suggested Tools for Taking Screenshots

- **Windows**: Use the Snipping Tool or press `Windows Key + Shift + S`
- **Mac**: Press `Command + Shift + 4`
- **Linux**: Use the built-in screenshot tool or press `PrtScn`

## Where to Save Screenshots

Save all screenshots in the `screenshots/` directory in the project root:

```
ai-quiz-generator/
├── screenshots/
│   ├── quiz_generation.png
│   ├── history_view.png
│   └── details_modal.png
```
